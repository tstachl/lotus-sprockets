__Lotus::Sprockets__ supports Ruby (MRI) 2+

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lotus-sprockets'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lotus-sprockets
